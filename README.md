# 🏠 Smart Home

Repositorio dedicado a mis proyectos de domótica, automatización y hogar inteligente.

Forma parte de **FJGR Labs** y recopila configuraciones, pruebas y desarrollos relacionados con Home Assistant, ESPHome, MQTT y dispositivos IoT.

---

## 🔧 Tecnologías

* Home Assistant
* ESPHome
* MQTT
* Zigbee
* Docker
* Linux
* ESP32

---

## 📂 Contenido

### Home Assistant Add-ons

Complementos desarrollados para ampliar funcionalidades de Home Assistant.

### Automatizaciones

Automatizaciones para mejorar la integración y el control del hogar.

### Dashboards

Interfaces y paneles personalizados.

### IoT Devices

Sensores, ESP32 y dispositivos conectados.

---

## 🚀 Objetivo

Experimentar y construir soluciones que permitan crear un hogar más inteligente, automatizado y eficiente.
